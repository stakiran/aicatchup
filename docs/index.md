# AIキャッチアップ

AI関連ニュース・論文を**週単位**で追いかける情報収集リポジトリ。
各ソースから、その週で最も反響・インパクトが大きいネタを最大5件ずつ拾って日本語でまとめます。

## 追っているソース

- [Anthropic ニュース](https://www.anthropic.com/news)
- [OpenAI ニュース](https://openai.com/news/)
- [Hugging Face 今週の論文](https://huggingface.co/papers/week)

## 週次ダイジェスト（新しい順）

- [2026年 第26週（6/22〜6/28）](weeks/2026-W26.md)
- [2026年 第25週（6/15〜6/21）](weeks/2026-W25.md)

## 使い方

Claude Code で `/catchup` を実行すると、上記ソースの新着を取得し、未取得のネタだけを当週のダイジェストに追記します。更新がなければ何もせず終了します。
